# react-django-spotify

A Full-stack Django &amp; ReactJS project that'll alow muliple people to connect to; they'll be able to control one spotify setup. Spotify's restful api will be used to do this. 

## to run

``` git clone https://github.com/keithmacinnis/react-django-spotify.git ```

navigate to ./spotify_controller/music_controller/front

``` npm i ``` 

``` npm run dev ```

navigate to ./spotify_controller/music_controller

``` python3 manage.py runserver ```

``` http://127.0.0.1:8000/ ``` should work.
